Урок 4. Проект Городская среда

Чтобы передать базу городских объектов в департамент полиции и завершить работу над прошлым заказом, нужно запрограммировать макеты городских пейзажей.
Заказчик выслал нам техническое задание.
Давайте изучим его!

Проект
...

Цель –– запрограммировать автоматическую отрисовку городской среды.

Задачи
...

  • Создать модуль city с функциями отрисовки городских объектов. Их должно быть не меньше трёх.
  • В отдельной программе нарисовать городской пейзаж.

Как и большинство заказчиков, представитель полиции не уточнил параметры объектов и их расположение.

Нужно придумать наполнение картины и спланировать свою работу самостоятельно.

Чем оригинальнее будет ваша картинка, тем лучше!

Чек-лист проекта "Городская среда"
...

Цель — запрограммировать картинку городской среды.
Требования:

  • наличие фона;
  • размещение не менее трёх разных объектов;
  • сохранение городских объектов в отдельном модуле city.

Чек-лист:

  • Внимательно изучить техническое задание.
  • Составить mind map проекта.
  • Запрограммировать городскую среду:
    • создать модуль с функцией-фоном и функциями-объектами города;
    • создать основную часть программы с отрисовкой городской среды.
  • Опубликовать проект в «Лаборатории».
  • Презентовать результат работы коллегам.
  • Собрать обратную связь коллег.

Основные команды
...

КомандаНазначение
forward(100)

Переместить черепашку вперёд на указанное число пикселей
(в примере 100)
left(90)

Повернуть черепашку влево на указанный угол в градусах
(в примере 90)
right(90)

Повернуть черепашку вправо на указанный угол в градусах
(в примере 90)
color("red")

Установить исполнителю новый цвет с указанным названием
(красный в примере)
pensize(5)

Изменить размер пера, которым рисует исполнитель
(изначально — 1, в примере 5)
circle(50)

Нарисовать круг с заданным радиусом в пикселях
(в примере 50)
begin_fill()

Начать заливку фигуры
(следом идёт команда рисования фигуры)
end_fill()

Завершить заливку фигуры
penup()

Поднять перо исполнителя
(полезно при рисовании нескольких фигур)
pendown()

Опустить перо исполнителя
goto(100, 100)

Переместить исполнителя в точку
с указанными координатами (в примере 100, 100)
speed(5)

Изменить скорость перемещения черепашки
(в примере 5)
hideturtle()

Скрыть исполнителя-черепашку с экрана
(остаётся только рисунок)
exitonclick()

Оставить изображение на экране
после исполнения программы
shape("turtle")

Изменить «костюм» черепашки (форму объекта).
Возможные костюмы:
"arrow", "turtle", "circle", "square", "triangle", "classic".
xcor()

Текущая координата Х черепашки (по горизонтали).
ycor()

Текущая координата Y черепашки (по вертикали).

from turtle import *

def sky():
    penup()
    goto(-250,-100)
    pendown()
    color('cyan')
    begin_fill()
    for i in range(2):
        forward(500)
        left(90)
        forward(250)
        left(90)
    end_fill()

def landscape():
    penup()
    goto(-250,-100)
    pendown()
    color('lime green')
    begin_fill()
    for i in range(2):
        forward(500)
        right(90)
        forward(100)
        right(90)
    end_fill()

sky()
landscape()

hideturtle()
exitonclick()

    pendown()
    color('lime green')

    end_fill()
